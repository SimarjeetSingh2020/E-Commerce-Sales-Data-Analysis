# E-Commerce-Sales-Data-Analysis

## Project Overview
This project analyzes e-commerce sales data to uncover trends, regional performance, and monthly sales behavior using Python.

## Tools Used
- Python
- Pandas
- Matplotlib

## Steps Followed
1. Data loading and cleaning
2. Exploratory data analysis
3. Visualization of insights
4. Report generation

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the script:
   python main.py

## Outputs
- Bar chart of sales by region
- Line chart of monthly sales trend
- Written business insights

## Conclusion
The analysis provides actionable insights to improve sales strategy and regional performance.
